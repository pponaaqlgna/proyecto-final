Vino Tinto


El vino tinto es de color rojo o morado, que puede variar según el tipo de uva y el tiempo de envejecimiento. Se elabora a partir de uvas tintas o rojas, que tienen la piel de color oscuro. Algunas de las variedades más comunes de uvas para vino tinto son el Cabernet Sauvignon, Merlot, Tempranillo, Malbec y Pinot Noir. 

Vino Blanco


  El vino blanco es de color dorado, amarillo pálido o incluso casi transparente, dependiendo de la uva y el proceso de vinificación. Se elabora principalmente con uvas blancas o verdes, como la Chardonnay, Sauvignon Blanc, Riesling y Pinot Grigio. Aunque a veces también se usan uvas tintas, siempre que se evite que las pieles entren en contacto con el mosto. 

Vino Rosa


El vino rosado tiene un color que varía desde el rosa pálido hasta el rosa intenso, dependiendo del tiempo de contacto con las pieles de las uvas. Generalmente se utiliza uva tinta para producir vino rosado, aunque también se pueden mezclar diferentes variedades. Algunas variedades comunes son Grenache, Syrah, Tempranillo y Pinot Noir.
